# 📊 ChurnWatch - Customer Churn Prediction System

## 📌 Project Overview

ChurnWatch is a Customer Churn Prediction System developed to identify customers who are likely to discontinue a subscription or service. The project combines data analysis, machine learning, SQL, and Power BI to help businesses understand customer behavior and improve customer retention.

---

## 🎯 Objective

The primary objective of this project is to:

- Predict customers who are at high risk of churn.
- Analyze customer behavior using historical data.
- Visualize important business insights.
- Support data-driven decision making for customer retention.
